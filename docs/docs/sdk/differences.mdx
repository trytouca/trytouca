```java
Touca.check("gpa", student.gpa, DecimalRule.absolute(x -> {
  x.setMin(3);
}));
```

With the Java SDK, you need to explicitly import the comparison rule that you
are using.

```java
import io.touca.rules.DecimalRule;
```

## Synchronous Comparison

Touca CLI commands `touca check` and `touca test` as well as the test runners
for Python and JavaScript SDKs have experimental support for synchronous
comparison. When enabled, as test cases are executed, the test output will print
whether the test results captured for a given test case match the baseline
version. Set configuration option `submission_mode` to `sync` to enable this
feature.

```plaintext
touca config set submission_mode=sync
```
